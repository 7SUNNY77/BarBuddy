# Датасет BarBuddy

В этой директории находится локальный каталог коктейлей для MVP.

## Файлы

- `cocktails.csv` — табличная версия каталога для анализа и обработки через Pandas
- `cocktails.json` — структурированная версия каталога для API
- `embeddings.npy` — векторные представления коктейлей, будет создан после подготовки описаний

## Источники

Основной источник — официальный каталог International Bartenders Association.

Перед публикацией каждая запись должна быть проверена вручную:
- Название коктейля
- Категория IBA
- Оригинальные ингредиенты и граммовки
- Способ приготовления
- Ссылка на первоисточник
- Право на использование изображения

Вкусовые теги и описание на русском языке создаются с помощью LLM и проверяются вручную.
